# 🚗 Vehicle Orders Data Analysis Dashboard (Power BI)

Interactive Power BI dashboard to analyze vehicle order trends, sales performance, and business insights.


## 🔹 Project Overview
This project presents an interactive dashboard built using Power BI to analyze vehicle order data. It focuses on identifying trends in sales, order distribution, and performance across different categories.


## 🔹 Objectives
- Analyze overall vehicle order trends  
- Identify top-performing categories/models  
- Track sales performance over time  
- Understand distribution across regions/customers  


## 🔹 Tools & Technologies
- Power BI  
- Data Cleaning & Transformation  
- Data Visualization  


## 🔹 Dashboard Features
- Interactive filters (Date, Category, Region, Model)  
- KPI cards for key metrics (Total Orders, Revenue, etc.)  
- Trend analysis over time  
- Category-wise and region-wise breakdown  


## 🔹 Key Insights
- Certain vehicle categories contribute the majority of orders  
- Sales trends show variation across time periods  
- Top models drive a significant portion of revenue  
- Regional distribution highlights key markets  


## 🔹 Files in Repository
- `Vehicle-Orders-Project.pbix` → Power BI dashboard  
- `Vehicle_Orders Folder` → Excel Datasets used
